# Đề Bài — session-01

## Problem Statement

Write a C program that:
1. Reads an integer from command line argument
2. Prints the square of that integer
3. Returns 0 on success

### Test Cases

- **TC-01-01:** Input: 5 → Expected output: "25"
- **TC-01-02:** Input: 0 → Expected output: "0"
- **TC-01-03:** Input: -3 → Expected output: "9"

## Requirements

- Use Linux Kernel Coding Style (tabs, K&R braces, 80-char lines)
- Use only standard C library
- Handle invalid input gracefully
